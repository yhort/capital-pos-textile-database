# capital-pos-textile-database
Módulo de base de datos para el sistema Capital POS (industria textil).  Incluye scripts de tablas, procedimientos almacenados, vistas y migraciones versionadas.

# Capital POS - Database

Base de datos del sistema Capital POS (industria textil).

## Estructura

- /tables → tablas base
- /procedures → procedimientos almacenados
- /views → vistas
- /migrations → cambios incrementales

## Flujo de trabajo

1. Crear rama feature/*
2. Crear o modificar scripts SQL
3. Hacer commit y push
4. Crear Pull Request
5. Revisar y hacer merge a main

## Requisitos

- SQL Server
- Acceso vía red (Tailscale)
